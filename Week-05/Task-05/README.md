# Week 5 Task 5 — Custom Object Detection Demo

## Overview

This project demonstrates a custom Computer Vision application capable of detecting:

* Person
* Car

A pre-trained YOLOv8 model was fine-tuned using a custom annotated dataset and then used for object detection on unseen test frames.

The final output was generated by performing detection on extracted frames and combining them into a smooth video with bounding boxes.

---

# Project Workflow

## 1. Dataset Preparation

* Extracted frames from video
* Created:

  * train dataset
  * validation dataset
* Annotated objects using Label Studio

Classes used:

* person
* car

---

# 2. Dataset Structure

```text
dataset/
│
├── train/
│   ├── images/
│   └── labels/
│
├── val/
│   ├── images/
│   └── labels/
│
├── test/
│   └── frames
│
├── data.yaml
├── train.txt
└── val.txt
```

---

# 3. Image Preprocessing

Frames were resized using ffmpeg while preserving aspect ratio.

Example command:

```bash
ffmpeg -i input.jpg -vf scale=384:-1 output.jpg
```

---

# 4. Model Training

YOLOv8 nano model was used for transfer learning.

Training command:

```bash
yolo detect train data=data.yaml model=yolov8n.pt epochs=100 imgsz=384
```

During training:

* training loss
* validation loss
* precision
* recall

were monitored to observe model learning and overfitting behaviour.

---

# 5. Generated Weights

Best trained weights:

```text
runs/detect/weights/best.pt
```

---

# 6. Detection on Test Frames

Detection command:

```bash
yolo detect predict model=C:\Users\apple\runs\detect\weights\best.pt source=test
```

Detected outputs were automatically saved in:

```text
runs/detect/predict/
```

---

# 7. Video Generation

Detected frames were converted into a smooth video using ffmpeg.

Command used:

```bash
ffmpeg -framerate 30 -i frame_%04d.jpg -c:v libx264 -pix_fmt yuv420p output.mp4
```

* inference using custom trained weights
