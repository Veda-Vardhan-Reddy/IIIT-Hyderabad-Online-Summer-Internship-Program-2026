# Week 5 — Computer Vision using Custom Object Detection

## Overview

Week 5 focused on building a complete Computer Vision object detection pipeline using transfer learning techniques.

The workflow included:

* dataset preparation
* image annotation
* preprocessing
* model training
* custom weight generation
* inference on unseen test data
* output video generation

A pre-trained YOLOv8 model was fine-tuned on a custom dataset containing:

* person
* car

classes.

The project involved annotating extracted video frames using Label Studio, resizing images while preserving aspect ratio, training a custom object detection model, and performing detection on unseen test frames.

The trained model successfully generated bounding-box based detections for persons and vehicles and demonstrated the practical application of deep learning in automated visual recognition systems.

This week provided hands-on experience in:

* dataset preparation
* transfer learning
* object detection
* training analytics
* inference pipelines
* video generation using Computer Vision techniques.
